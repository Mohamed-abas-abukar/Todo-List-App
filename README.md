# Todo-List-App
A simple and interactive Todo List Application built using HTML , CSS , Vanilla JavaScript . The app allows users to add , complete and delete tasks with clean and user-friendly interface . This project focuses on practicing DOM manipulation, event handling , and basic UI design with out using any external libraries or framework

# Features 
. Add new task
. Delete individual tasks
. Clear all tasks 
. Simple and moder UI design 

# Teachnology  
. HTML 
. CSS 
. JavaScript 

# What i learn
. Working with DOM dynamically 
. Handling user events (click , input ) 
. Creating and removing element using Javascript 
. Improving UI/UX with Css style
